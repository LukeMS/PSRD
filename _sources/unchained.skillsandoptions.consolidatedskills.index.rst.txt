Consolidated Skills
####################

The skills in the \ *Core Rulebook*\  allow for complex specialization. In some games, the GM might wish to use a smaller list to make characters more broadly talented, to group skills that characters typically choose together, and to speed up the leveling-up process. The consolidated skills system reduces the number of skills by combining related skills. In many cases, the new skills have been renamed to highlight the nature of the consolidated skill. This variant system makes each skill rank matter more. Even after reducing the number of skill ranks granted, characters will be somewhat more skilled. Character classes that are highly skilled, such as the bard or rogue, get the biggest boost.

The consolidated skills system reduces the number of skills from 35 to 12. Many knowledge skills have been placed into larger categories with a similar theme. :ref:`Knowledge <corerulebook.skills.knowledge#knowledge>`\  (arcana), for example, is combined with :ref:`Use Magic Device <corerulebook.skills.usemagicdevice#use_magic_device>`\  under :ref:`Spellcraft <corerulebook.skills.spellcraft#spellcraft>`\ . The list of new skills appears on :ref:`Table: Consolidated Skills <unchained.skillsandoptions.consolidatedskills.index#consolidated_skills>`\ , along with the key ability for each consolidated skill and the core skills it replaces. Several skills are removed and not replicated by this system, as indicated by a dash (—) on :ref:`Table: Converting Core Skills <unchained.skillsandoptions.consolidatedskills.index#converting_core_skills>`\ . These are typically skills that are less important for adventuring, but can be put back into your game using the :ref:`background skills variant <unchained.skillsandoptions.backgroundskills>`\ . Skills can be converted to and from the consolidated skills system using :ref:`Table: Converting Core Skills <unchained.skillsandoptions.consolidatedskills.index#converting_core_skills>`\ . This is most helpful when running published adventures or using monsters built using the core system.

Monster Identification
=======================

You can use certain skills to identify monsters and their special powers or vulnerabilities. A successful check allows you to remember a bit of useful information about that monster. For every 5 points by which your check result exceeds the DC, you recall another piece of useful information.

.. list-table::
   :header-rows: 1
   :class: contrast-reading-table
   :widths: auto

   * - Monster Rarity
     - Skill Check DC
   * - Common (goblin)
     - 5 + monster's CR
   * - Most monsters
     - 10 + monster's CR
   * - Rare (tarrasque)
     - 15 + monster's CR

.. _`unchained.skillsandoptions.consolidatedskills.index#recall_knowledge`:

Recall Knowledge
=================

Some skills have the recall knowledge function. This lets you answer questions about the subjects in question. This has a DC of 10 (for really easy questions), 15 (for basic questions), or 20 to 30 (for really tough questions).

\ **Action**\ : Usually none. Using a library (see Untrained, below) takes 1d4 hours.

\ **Untrained**\ : You can't attempt an untrained knowledge check with a DC higher than 10 unless you have access to an extensive library on the subject.

.. _`unchained.skillsandoptions.consolidatedskills.index#racial_bonuses`:

Racial Bonuses
***************

The races of the \ *Core Rulebook*\  that gain bonuses to skills should change in the following ways.

\ **Dwarf**\ : Greed functions the same way if the campaign uses background skills; otherwise, its bonus instead applies to :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks to identify magic items that contain precious metals or gemstones.

\ **Elf**\ : Keen senses grants a +1 bonus on :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks.

\ **Gnome**\ : Keen senses grants a +1 bonus on :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks. Obsessive functions the same way if the campaign uses background skills; otherwise, it instead grants a +1 bonus on the character's choice of :ref:`Perform <corerulebook.skills.perform#perform_skill>`\  or :ref:`Spellcraft <corerulebook.skills.spellcraft#spellcraft>`\  checks.

\ **Half-Elf**\ : Keen senses grants a +1 bonus on :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks.

\ **Half-Orc**\ : Intimidating applies only to Influence checks to demoralize or intimidate.

\ **Halfling**\ : Keen senses grants a +1 bonus on :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks. Sure-footed grants a +1 racial bonus on :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  and Athletics checks.

.. _`unchained.skillsandoptions.consolidatedskills.index#converting_existing_characters`:

Converting Existing Characters
===============================

When converting to the consolidated skills system in the middle of an ongoing campaign, it's typically best to allow each player to completely reset her character's skill ranks and spend them again from scratch. She should also be able to retrain any feats or other abilities that affect her skill bonuses.

.. _`unchained.skillsandoptions.consolidatedskills.index#altering_skill_bonuses`:

Altering Skill Bonuses
***********************

To adjust the rest of the system, use the guidelines below. Generally, an ability or effect that would normally grant a bonus on multiple types of skill checks instead grants a bonus on half as many, rounded down (minimum 1).

.. _`unchained.skillsandoptions.consolidatedskills.index#bonuses_for_function`:

Bonuses for Function
=====================

For feats and abilities that grant bonuses or penalties to skills, it's best for the GM to either ban them or adjust them to work only with specific functions of the feat. That is to say, the consolidated skills still cover the functions of the skills they've absorbed, so a bonus can still apply to only a specific function. For example, the Intimidating Prowess feat should add a character's Strength modifier instead of Charisma only when using the :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  skill to intimidate, rather than adding it to the :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  skill in all cases. The \ *jump*\  spell should give a bonus only on :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks to jump, rather than on all :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks. There are no direct conversions for this effect, so it must be done on a case-by-case basis.

.. _`unchained.skillsandoptions.consolidatedskills.index#prerequisites`:

Prerequisites
==============

If a feat, prestige class, or other ability requires a certain number of ranks in a skill, directly convert that core skill to the consolidated skill. Use the same number of ranks. For example, the Mounted Combat feat would require :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  1 rank instead of :ref:`Ride <corerulebook.skills.ride#ride>`\  1 rank.

.. _`unchained.skillsandoptions.consolidatedskills.index#monster_skill_bonuses`:

Monster Skill Bonuses
======================

When running a prebuilt monster, the GM can convert core skills to consolidated skills on the fly. The GM can choose one of two paths: either use the monster's highest printed bonus as its bonus with the relevant consolidated skill, or treat each bonus as though it applies only to a sub-function of a consolidated skill. For example, an ice devil has a +22 bonus on :ref:`Acrobatics <corerulebook.skills.acrobatics#acrobatics>`\  checks and a +13 bonus on :ref:`Fly <corerulebook.skills.fly#fly>`\  checks. The GM could either give the devil a +22 bonus on all :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics>`\  checks, or use the +22 bonus if the monster is moving through a threatened area and the +13 bonus if it's flying. Either approach can work, but the GM should be consistent with which she uses.

A GM using the highest-bonus approach still might want to consider separating out the bonuses in the case of large racial modifiers. A monster with a climb speed should probably not get its +8 racial bonus on Athletics checks to jump or swim.

.. _`unchained.skillsandoptions.consolidatedskills.index#altering_feats`:

Altering Feats
***************

Feats that give bonuses on skill checks need to be adjusted or removed in the consolidated skills system. This section presents a list of the skill feats in the \ *Core Rulebook*\  that shouldn't be used with this system. It also presents an alternative version of Skill Focus that replaces the core version to make its power level work better with this system. If an ability or class feature would grant access to one of the banned feats, use the most appropriate Skill Focus feat instead, as determined by the GM. For example, a familiar that grants Alertness would instead grant Skill Focus (:ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\ ). For feats that modify skills in other ways, see the :ref:`Bonuses for Function <unchained.skillsandoptions.consolidatedskills.index#bonuses_for_function>`\  section above.

\ **Removed Feats**\ : Acrobatic, Alertness, Animal Affinity, Athletic, Deceitful, Deft Hands, Magical Aptitude, Persuasive, Self-Sufficient, Stealthy.

.. _`unchained.skillsandoptions.consolidatedskills.index#skill_focus_unchained`: `unchained.skillsandoptions.consolidatedskills.index#skill_focus`_

.. _`unchained.skillsandoptions.consolidatedskills.index#skill_focus`:

Skill Focus
============

Choose a skill. You are particularly adept at that skill.

\ **Benefit**\ : You gain a +2 bonus on all checks involving the chosen skill. If you have 10 or more ranks in that skill, this bonus increases to +4.

\ **Special**\ : You can gain this feat multiple times. Its effects do not stack. Each time you take the feat, it applies to a new skill.

.. _`unchained.skillsandoptions.consolidatedskills.index#altering_traits`:

Altering traits
****************

Many traits offer a +1 trait bonus on core skill checks and make those skills class skills. With the consolidated skills system, those traits instead grant a +1 trait bonus on traits involving the functions of those consolidated skills corresponding to the traits' listed core skills, or a +4 trait bonus if the skills are not class skills. If you later receive that skill as a class skill, this trait bonus reduces to +1. 

.. _`unchained.skillsandoptions.consolidatedskills.index#special_class_rules`:

Special Class Rules
********************

Some class features function differently because of the consolidated skills list. Clarifications appear here, divided by class. These entries cover only the main class features, so the GM might need to adjudicate how certain arcane schools, domains, and the like interact with the consolidated skills system. Class features are omitted if they can be handled with a simple substitution, such as changing an ability that requires a :ref:`Disable Device <corerulebook.skills.disabledevice#disable_device>`\  check to instead require a Finesse check.

\ **Alchemist**\ : Because the consolidated skills system removes :ref:`Craft <corerulebook.skills.craft#craft>`\  skills by default, the GM should either treat the alchemist's :ref:`Craft (alchemy) <corerulebook.skills.craft#craft>`\  bonus as though it were equal to his character level + his Intelligence modifier + 3 or use the :ref:`background skills <unchained.skillsandoptions.backgroundskills>`\  system.

\ **Barbarian**\ : The :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  skill and the intimidate function of :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  can be used while raging.

\ *Rage Powers*\ : Raging climber, raging leaper, and raging swimmer are replaced with raging athlete, which adds the barbarian's level as an enhancement bonus on all :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks and treats all such checks to jump as though the character had a running start.

\ **Bard**\ : Note that :ref:`Performance <unchained.skillsandoptions.consolidatedskills.performance#performance_skill>`\  applies to all types of performance, rather than being broken down like the core :ref:`Perform <corerulebook.skills.perform#perform_skill>`\  skills. Bardic knowledge adds 1/2 the bard's class level (minimum 1) on all :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks, and the bard can attempt checks with these skills untrained. Versatile performance allows the bard to substitute :ref:`Performance <unchained.skillsandoptions.consolidatedskills.performance#performance_skill>`\  when attempting either :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  or :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks. He must choose only one of these two skills, and doesn't increase the number of substituted skills at higher levels. Lore master applies to :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks.

\ **Bloodrager**\ : The :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  skill and the intimidate function of :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  can be used while bloodraging.

\ **Cavalier**\ : A cavalier order adds only one skill to the cavalier's list of class skills (of the GM's choice, but the choice can't overlap with the cavalier's normal class skills). The bonus on checks applies only to the specific skill function listed.

\ **Druid**\ : Nature sense instead grants a +1 bonus on :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\  and :ref:`Survival <unchained.skillsandoptions.consolidatedskills.survival#survival_skill>`\  checks.

\ **Hunter**\ : The bonus from the monkey aspect applies only to :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks to climb.

\ **Inquisitor**\ : Stern gaze grants the inquisitor a bonus on :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks to intimidate and on :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks to sense motive.

\ **Investigator**\ : Because the consolidated skills system removes :ref:`Craft <corerulebook.skills.craft#craft>`\  skills by default, the GM should either treat the investigator's :ref:`Craft <corerulebook.skills.craft#craft>`\  (alchemy) bonus as though it were equal to his character level + his Intelligence modifier + 3 or use the :ref:`background skills <unchained.skillsandoptions.backgroundskills>`\  system. The investigator's ability to apply inspiration without expending uses applies to :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks (he must still be trained in the skill). The trapfinding bonus applies to both :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks to locate traps and :ref:`Finesse <unchained.skillsandoptions.consolidatedskills.finesse#finesse_skill>`\  checks to disable traps. An investigator can use :ref:`Finesse <unchained.skillsandoptions.consolidatedskills.finesse#finesse_skill>`\  to disable magical traps. Keen recollection applies to :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks.

\ *Investigator Talents*\ : Device talent allows the investigator to use :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  untrained, but has no effect if the investigator is trained in :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\ . Empathy applies only when the investigator uses :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  to sense motive. Expanded inspiration can be used with :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  and :ref:`Survival <unchained.skillsandoptions.consolidatedskills.survival#survival_skill>`\  checks. Inspirational expertise applies when the investigator identifies a monster's special powers or vulnerabilities using whichever skill is appropriate. Inspired intelligence applies to :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks. Underworld inspiration applies to :ref:`Finesse <unchained.skillsandoptions.consolidatedskills.finesse#finesse_skill>`\  and :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks.

\ **Oracle**\ : An oracle gains two class skills from her mystery (chosen by the GM).

\ **Ranger**\ : The favored enemy bonus applies to :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  and :ref:`Survival <unchained.skillsandoptions.consolidatedskills.survival#survival_skill>`\  checks against creatures of the selected type, and to checks with the monster knowledge function of the relevant skill. The favored terrain bonus applies to initiative checks, :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\  checks, :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks to notice things, :ref:`Stealth <unchained.skillsandoptions.consolidatedskills.stealth#stealth_skill>`\  checks, and :ref:`Survival <unchained.skillsandoptions.consolidatedskills.survival#survival_skill>`\  checks.

\ **Rogue**\ : The trapfinding bonus applies to both :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks to locate traps and :ref:`Finesse <unchained.skillsandoptions.consolidatedskills.finesse#finesse_skill>`\  checks to disable traps. A rogue can use :ref:`Finesse <unchained.skillsandoptions.consolidatedskills.finesse#finesse_skill>`\  to disable magical traps.

\ *Rogue Talents*\ : The skill mastery talent applies to a number of skills equal to 1 + 1/2 the rogue's Intelligence modifier. It can still be taken multiple times.

\ **Skald**\ : Note that :ref:`Performance <unchained.skillsandoptions.consolidatedskills.performance#performance_skill>`\  applies to all types of performance, rather than being broken down like the core :ref:`Perform <corerulebook.skills.perform#perform>`\  skills. Bardic knowledge adds 1/2 the skald's class level (minimum 1) on all :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks, and the skald can attempt checks with these skills untrained. The :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  skill and the intimidate function of :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  can be used while in an inspired rage. Versatile performance allows the skald to substitute :ref:`Performance <unchained.skillsandoptions.consolidatedskills.performance#performance_skill>`\  when attempting either :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks or :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks to sense motive. He must choose only one of these two skills, and doesn't increase the number of skills substituted at higher levels. Lore master applies to :ref:`Nature <unchained.skillsandoptions.consolidatedskills.nature#nature_skill>`\ , :ref:`Religion <unchained.skillsandoptions.consolidatedskills.religion#religion_skill>`\ , :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\ , and :ref:`Spellcraft <unchained.skillsandoptions.consolidatedskills.spellcraft#spellcraft_skill>`\  checks.

\ **Slayer**\ : The studied target bonus applies to :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks to bluff, :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  checks, and :ref:`Survival <unchained.skillsandoptions.consolidatedskills.survival#survival_skill>`\  checks against the studied target. The stalker bonus applies to :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks to intimidate a creature, :ref:`Perform <corerulebook.skills.perform#perform_skill>`\  checks to use a disguise, and :ref:`Stealth <unchained.skillsandoptions.consolidatedskills.stealth#stealth_skill>`\  checks.

\ *Slayer Talents*\ : Foil scrutiny applies to :ref:`Influence <unchained.skillsandoptions.consolidatedskills.influence#influence_skill>`\  checks to bluff a creature and :ref:`Perform <corerulebook.skills.perform#perform_skill>`\  checks to use a disguise. Trapfinding grants :ref:`Finesse <unchained.skillsandoptions.consolidatedskills.finesse#finesse_skill>`\  as a class skill. \ **Sorcerer**\ : A bloodline's class skill applies to all uses of the corresponding consolidated skill.

\ **Summoner**\ : Change the eidolon's class skills to :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  and :ref:`Stealth <unchained.skillsandoptions.consolidatedskills.stealth#stealth_skill>`\ . \ **Swashbuckler**\ : Derring-do can be used on :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  and :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks. Swashbuckler's edge can be used on :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\  and :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks.

\ **Witch**\ : The bonus from the flight hex is a +4 racial bonus on :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  checks to swim.

Miscellaneous Abilities
========================

Abilities that aren't tied only to one class appear here.

\ **Animal Companions**\ : Change animal skills to :ref:`Acrobatics <unchained.skillsandoptions.consolidatedskills.acrobatics#acrobatics_skill>`\ , :ref:`Athletics <unchained.skillsandoptions.consolidatedskills.athletics#athletics_skill>`\  \*, :ref:`Perception <unchained.skillsandoptions.consolidatedskills.perception#perception_skill>`\  \*, :ref:`Stealth <unchained.skillsandoptions.consolidatedskills.stealth#stealth_skill>`\  \*, and :ref:`Survival <unchained.skillsandoptions.consolidatedskills.survival#survival_skill>`\ . An asterisk ( \*) indicates a class skill.

\ **Familiars**\ : The skill bonus from the familiar applies to all uses of the corresponding consolidated skill, but the bonus is reduced to +2. The raven familiar grants a +2 bonus on :ref:`Society <unchained.skillsandoptions.consolidatedskills.society#society_skill>`\  checks.
